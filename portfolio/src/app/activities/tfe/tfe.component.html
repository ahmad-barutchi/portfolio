Résumé

Le but du projet est la mise en oeuvre du robot et d'une application web et mobile pour la surveillance (monitoring) des données générer par le robot (IOT).

L'application est basé sur le template de monitoring Ngx-Admin

Partie Software

Ngx-admin

Afin de développer notre application web, il a été important de réfléchir aux outils, framework à utiliser. Il existe plusieurs solutions de monitoring avec lesquelles nous pouvons travailler et développer notre propre solution. Afin de ne pas réinventer la roue, Akveo propose des tableaux de bord d’administrateur qui vont nous permettre de faire du monitoring de données et du big data, nous aurons la possibilité d’implémenter n’importe quel type de composants à notre tableau de bord.
Ngx-admin est un modèle Angular créer par Akveo, ce tableau de bord très connu est open source et gratuit. Il est construit sur Angular 13+ en prenant en charge Eva Design System et Nebular.

Visualisation

Ceci est un graphe pour comparer les valeurs prise par le robot (témperature avant, témperature arrière, témperature ambiante, et la humidité) ainsi que les courbes de prédiction d'une présance vivante ou d'un potentiel source de feu/innondation.
les boutons coloré sert à filtrer/supprimer un courbe pour mieux visualiser les autres.

Les données sont affichées également dans un tableau pour permetre de faire des recherche par catégories ou valeurs. Afin de savoir directement si nous avons la présence d’un être humain ou d’un feu directement sans devoir faire une analyse par graphique nous centralisons les informations dans un tableau.

L’utilisateur doit principalement se concentrer sur la colonne « Origin » afin de vérifier s’il s’agit d’un être humain ou d’un feu, toute la logique est en place dans le Backend. Ensuite, il lui suffit de regarder la position GPS, de regarder l’heure de détection et enfin d’analyser les probabilités. Une probabilité plus élevée devra être prise en charge en priorité des autres détections

dans cette example nous pouvons manipuler les différents séances ou les supprimer
